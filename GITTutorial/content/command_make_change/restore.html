Official Git Documentation - Restore

Restore (AKA 'undo changes')

DETAILS

Performing an 'restore' is the process of updating the with the contents of the remote source.

Note: 'Restoring' a deleted file is the same as un-deleting it.

git restore .

'Un-does' all changes to files / folders.

git restore *.html

'Un-does' changes done to files with an extension of 'html'.
