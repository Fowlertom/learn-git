Official Git Documentation - Remove

Remove (AKA 'unstage')

DETAILS

Performing a 'remove' is the process of removing a file from the index, but not removing it from your working folder.

git rm -f --cached [FILENAME]

Removes [FILENAME] from being but does not delete it.

-f force removal

--cached do not delete file after it is unstaged

git rm -f --cached *.html

Removes 'staged' files wtih an extension of 'html'.
