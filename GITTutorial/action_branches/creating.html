Official GIT documentation - Branching

Creating / Deleting Branches

INTRODUCTION

To add/edit any content, local branch must be created.

DETAILS

ACTION:

Create Branch
git branch <branch>

Creates local branch.

ACTION:

Create Branch (linked to remote)
git branch -f <branch>

Creates local branch. Will be 'linked' to a remote branch with the same name.

ACTION:

Delete Branch
git branch -D <branch>

Deletes a local branch. Will be deleted if it has unmerged changes.This is the transactional equivalent of:

$ git branch -d -f <branch>(1)

  1. forces a branch to be reset to it's starting point, then deletes it
