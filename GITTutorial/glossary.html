Glossary (brief)

A complete glossary is located in the Git User's Manual

Term Description
gitan open source, distributed version-control system
GitHub a platform for hosting and collaborating on Git repositories
commit a Git object, a snapshot of your entire repository compressed into a SHA
branch a lightweight movable pointer to a commit
clone a local version of a repository, including all commits and branches
remote a common repository on GitHub that all team member use to exchange their changes
fast-fowrard merge head of the current branch is moved forward to point at the head of the merged-in branch, without a 'merge commit' being created
pull request a place to compare and discuss the differences introduced on a branch with reviews, comments, integrated tests, and more
origin remote repository that a project was originally cloned from. Used instead of that original repository's URL - making referencing easier
HEAD representing your current working directory, the HEAD pointer can be moved to different branches, tags, or commits when using git checkout
porcelain command high level git command
plumbing command low level git command
ahead a (local or remote) branch has more recent commits that it's associated (local or remote) branch
behind opposite of ahead
