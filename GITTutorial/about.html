About James' GIT Tutorial

Created by me (James Fowler) to teach myself GIT.

This website is stored in GitHub, thereby forcing me to learn GIT in order to store the website artifacts.

This technique (forcing a tutorial about GIT to be stored in GitHub) is a variant on dogfooding.

This web-site was created using the following technologies:

  • HTML (of course)
  • jQuery
  • BootStrap (NavBar)
  • Font-Awesome 4.70
  • Google Fonts (Droid Sans Mono, Noto Serif, Open Sans)
  • CSS Styles copied from the GIT for Windows Tools
