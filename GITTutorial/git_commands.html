Git Commands

This page focuses on Git commands that are performed on your . The most 'common' usage is listed first for each topic. Note: the categories were copied from the GitHub Cheat Sheet

Branches

Create and Switch - create, and switch to branch

Switch - switch to branch

Delete - delete branch

Create - create branch

Synchronize Changes

Rebase - apply changes to a branch from a different (base) branch

Pull - fetch changes from a remote branch, merge changes into local branch

Push - update a remote branch with the commits from a local branch

Make Changes

Add - prepare (aka 'stage') changes to be applied to a branch

Commit - apply changes to a branch

Undo/Discard Changes

Reset - prepare (aka 'stage') changes to be applied to a branch

Clean - remove untracked changes from the working tree

Create Repositories

Clone - create a local repository
