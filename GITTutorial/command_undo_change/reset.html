Official Git documentation - Reset

Reset (aka Undo)

DETAILS

Performing a 'reset' is the process of returning a branch to a state before any merges. If you are 'resetting' from origin, you are undoing all local commits - effectively if you delete the working tree and get a new copy.

git reset --hard <Origin/feature_branch>

Discards any commits on the current branch in your working tree and makes the current branch identical to <Origin/feature_branch>.
