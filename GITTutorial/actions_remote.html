Remote Actions

This page focuses on actions / commands that are performed in a browser on Git repositories hosted in Team Foundation Server (TFS). Listed from highest-to-lowest frequency.

Create pull request

  1. Open a browser. Navigate to the TFS project hosting the repository
  2. Select the project you want to add the repository for
  3. Click the '+' (plus) icon beside the project name
  4. Select 'new repository'
  5. Enter the 'Repository Name', then click 'create'

Create a new branch from a <base_branch>

  1. Open a browser. Navigate to the TFS project hosting the repository
  2. Click on 'Repos' and select the repository
  3. Click on 'branches'
  4. On the right hand panel, click the '...' beside <base_branch> and select 'New Branch'.
  5. Enter the branch name.
  6. Setup branch policies as needed

Create a new repository

  1. Open a browser. Navigate to the TFS project hosting the repository
  2. Select the project you want to add the repository for
  3. Click the '+' (plus) icon beside the project name
  4. Select 'new repository'
  5. Enter the 'Repository Name', then click 'create'

Merge <source_branch> to <target_branch>

  1. Open a browser. Navigate to the TFS project hosting the repository
  2. Click on 'Repos' and select the repository
  3. Click on 'branches'
  4. On the right hand panel, click the '...' beside <target_branch> and select 'set as compare branch'. Unlock branch if necessary.
  5. Hover of the line of the <source_branch>, select 'new pull request'.
  6. Enter required fields, then click 'create'
