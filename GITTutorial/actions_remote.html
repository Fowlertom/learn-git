Create a new repository
  1. Open a browser. Navigate to the TFS project hosting the repository
  2. Select the project you want to add the repository for
  3. Click the '+' (plus) icon beside the project name
  4. Select 'new repository'
  5. Enter the 'Repository Name', then click 'create'
