Switching Between Branches

SUMMARY

To add/edit content, a branch must 'have focus'.

DETAILS

ACTION:

Switch Branch
git checkout <branch>

If <branch> exists (with the same name) in the remote, it will be 'linked' to that remote branch.

ACTION:

Create and Switch Branch (using an existing branch as a 'base')
git checkout -b <new_branch> <base_branch>

<base_branch> must exist locally. This is the transactional equivalent of: 

$ git branch -f <new_branch> [<start point>]
$ git checkout <new_branch>
