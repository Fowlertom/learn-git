Merge

DETAILS

Performing a "merge" is the process of bringing the contents of another branch into the current branch.

git merge <branch_with_changes>

Moves the 'commits' of the <branch_with_changes> to <current_branch>.

EXAMPLE

Assume the following history exists:

          A---B---C master
         /
    D---E---F---G feature_branch

From this point, the result of the following commands:

git checkout target_branch
git merge rebase master

would be:

          A---B---C master
         /         \
    D---E---F---G---H feature_branch
